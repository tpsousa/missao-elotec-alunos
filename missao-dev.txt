# A Missão dos Desenvolvedores Perdidos

Era uma vez uma equipe de desenvolvedores encarregada de entregar o projeto mais importante do ano.  
O sistema precisava estar pronto em apenas duas semanas, mas havia um problema: os requisitos mudavam toda hora, e os bugs surgiam do nada.

O líder do time, chamado Alex, decidiu que eles precisavam de uma nova estratégia.  
Era hora de trabalhar em equipe, dividir tarefas, controlar versões e agir com disciplina.

Foi então que surgiu uma ideia: usar uma ferramenta chamada Git para gerenciar tudo!  
Mas, assim que começaram a usar o Git, novos desafios surgiram...

(continuação pelos desenvolvedores 👨‍💻👩‍💻: cada aluno deve adicionar um novo parágrafo contando o que aconteceu a seguir na missão!)
